Además le asigna estilos a los elementos de manera prederminada. los parrafos "p" le coloca una identación, las etiquetas hr le da la apariencia que vemos en la linea que esta arriba, lo input, textareas, a y otros elementos son transformado por el tema establecido. Para establecer un tema solo deben colocar en la etiqueta HTMl data-themen="tutema" es caso de omitirlo le pondra el tema shell que es el que vemos en esta pagina. si deseamos crear un nuevo tema en la carpeta de Framework encontras un carpeta llamada css3 en ella crea una carpeta con el nombre de tu tema y adentro establece el archivo pricipla el cual se debe llamar ShellScript.css y dentro podras incluir tus hojas de estilos con @import url('style.css'); luego debes colocar en el HTML data-themen="nombre_de_la_carpeta" y listo tu página trabajara con un nuevo tema css.

También se posee funciones para trabajar con estilos y apariencia las cuales veremos a continuación:

Se usa para asignar o obtener los estilos css de un elemento. 

//Sintasis
S("mielemento").stylo(propety, value);
//Usos
//Establecer estilos
S("mielemento").stylo("propiedad", "valor");
S("mielemento").stylo({
	propiedad:valor
});
/*
 * en la primera se asigna solo una propiedad y su valor
 * el la segunda se envia un objeto de propiedad y valores css
 */

//Obtener estilos
S("mielemento").stylo();
S("mielemento").stylo("propiedad");
/*
 * en la primera se obtinen todos las propiedades css que posee el elemento 
 * el la segunda se obtine el valor de la propiedad pasada 
 */

Se usa para eliminar una o varias propiedades css 

//Sintasis
S("mielemento").removeStylo(propety);
//Usos
//Remover estilos
S("mielemento").removeStylo("propiedad");
S("mielemento").removeStylo([
	"propiedad1","propiedad2"
]);
/*
 * en la primera se asigna solo una propiedad y su valor
 * el la segunda se envia un objeto de propiedad y valores css
 */

Se usa para obtener o establecer el ancho y alto de un elemeto según la función 

//Sintasis
S("mielemento").ancho(valor);
S("mielemento").alto(valor);
//Usos
//Establecer
S("mielemento").ancho(valor);
S("mielemento").alto(valor);
//Obtiene
S("mielemento").ancho();
S("mielemento").alto();

Se usa para obtener o establecer una posición a un elemento dado. 

//Sintasis
S("mielemento").posicion(tipo,left,top);
//Usos
//Establecer
S("mielemento").posicion("Tipo","left","top");
S("mielemento").posicion({
	position:"tipo",
	top:"top"
	left:"lef"
});
/*
 * se puede establecer la posicion ya sea por la primera forma
 * la segunda la diferencia es que el la segunda puede establecer uno
 * de los 3 parametros y en la primera debes establecerlos todos
 */

//Otener
S("mielemento").posicion();
//retorna un objeto con la información de la posición del elemento
