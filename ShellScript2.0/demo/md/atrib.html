Atributos y Clases

ShellScript nos facilita la vida a la hora de trabajar con atributos y clases posee varias funciones poderoras que se encargan de todo eso y mas.

Atributos

Para manipurlar cualquier atributo ShellScript nos ofrese 2 funciones atrib y removeAtrib.

Se usa para asignar y buscar atributos de un elemento dado. 

//Sintasis
S("mielemento").atrib(attr, valor);
//Usos
//Establecer Atributos
S("mielemento").atrib("miatributo", "mivalor");
S("mielemento").atrib({
	miatributo:"mivalor",
	miatributo:"mivalor"
});
//en la primera establese solo un atributo y el la segunda varios atributos
//Buscar Atributos
var a =  S("mielemento").atrib("miatributo");
//la variable a ahora tiene el valor del atributo o null de no existir

Se usa para remover un atributo o varios atributos de un elemento dado. 

//Sintasis
S("mielemento").removeAtrib(attr);
//Usos
//Remover Atributos
S("mielemento").removeAtrib("miatributo");
S("mielemento").removeAtrib([
	"miatributo",
	"miatributo",
	"miatributo",
	"miatributo"
]);
/*
 * en la primera remueve solo un atributo y 
 * el la segunda remueve todos los atributos del array
 */

Data

Existe el atributo data el cual nos permite crear nuestros propios atributos por ejemplo data-algo, data-rol etc... Para ellos ShellScript desarrollo 2 funciones para trabajar netamente con estos data y removeData.

Se usa para estableser o buscar un atributo data-"algo". 

//Sintasis
S("mielemento").data(key,valor);
//Usos
//Establecer Atributos
S("mielemento").data("algo", "mivalor");
S("mielemento").data({
	"algo":"valor"
});
/*
 * en la primera se crea el atributo data-algo
 * el la segunda se crean varios atributo data-algo
 * como pueden notar no se coloca data en el atributos
 * solo lo que va despues de guion ejemplo
 * data-rol se coloca rol
 */

//Obtener valor Atributos
var a =  S("mielemento").data("algo");
//la variable a contiene el valor del atributo data-algo

Se usa igual que removeAtrib y tienes 2 formas de darles el atributo puedes darle data-rol o solo rol ya sea en un array o string. 

S("mielemento").removeData("algo");
S("mielemento").removeData([
	"data-rol",
	"rol"//Es igual al anterior
]);

Clases

Otro atributo que se usan son el trabajo de clases. Para ello tenemos multiples funciones y una de ellas la lo habiamos explicado en el apartado Funciones / Validadores / Classexiste

Se usa para asignar una clase al elemento dado. 

S("mielemento").AddClass("class");

Se usa para remover una clase al elemento dado. 

S("mielemento").RemoveClass("class");

Se usa para asignar o remover una clase al elemento dado si el elemento posee la clase se la elimina en su defecto se la asigna 

S("mielemento").toggleClass("class");
