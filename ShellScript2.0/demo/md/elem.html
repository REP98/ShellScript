Trabajo con Elementos

Poseemos funciones para agregar elementos, copiarlos, moverlos, clonarlos, herencias y removerlos veremos a continuación cuales son:

Para saber cual es el padre de un elemento ShellScript no posee ningna función ya que es un metodo establecido al seleccionar un elemento en si 

//Al hacer esto
var a = S("mielemento");
/*
 * la variable a es un objeto que contiene
 * length      cantidad de elementos seleccionados
 * padre       es el padre del elemento selecionado
 * selector    es el string o texto escrito entre los parentesis en 
 *             este caso mielemento
 * thisElement el elemento o array de elementos selecionados
 * asi que si deseamos el padre solo devemos colocar lo siguiente
 */
S("mielemento").padre;
//o
a.padre;
abuelo
Busca el abuelo del elemento dado y retorna el elemento que es abuelo o null 
var a = S("body").abuelo();
//esta variable tiene al abuelo de body que es document
hijo
Busca todos los hijos del elemeto retorna un HTMLCollection o null
var a = S("body").hijo();
//esta variable tiene a todos los hijos de body en forma de array
prihijo
Obtine el primer hijo del elemeto retorna el primer hijo o null
var a = S("body").prihijo();
ulhijo
Obtine el último hijo del elemeto retorna el último hijo o null
var a = S("body").ulhijo();
Sigher
Obtine el siguiente hermano del elemeto es decir el que esta despues del elemento dado retorna el hermano o null
var a = S("body").Sigher();
Prevher
Obtine el hermano anterior del elemeto es decir el que esta despues del elemento dado retorna el hermano o null
var a = S("body").Prevher();

Para crear nuevos elemetos con ShellScript es muy sencillo ya que como se selecciona un elemento se le puede pasar un segundo argumento que son parametros de creación. 

S("div",{
	propiedades
})

De esta forma hemos creado un nuevo elemento y aun no lo hemos asignado al DOM, las propiedades son funciones de ShellScript o de Javascript pasadas como metodos. 

var element =S("div",{
	html:"Soy un div"
})
//que es lo mismo
S("div").html("Soy un div");

La diferencia entre ambos es que con el primero estamos creando un nuevo div y con el segundo estamos dandole un texto a un div existente.

las funciones para agregar nuevos elementos son las siguiente:
Veremos la varible element que representa el nuevo elemento que creamos en el ejemplo anterior

antes
agrega el nuevo elemento antes de un elemento dado
element.antes("h3" //selector o elemeto);
despues
agrega el nuevo elemento despues de un elemento dado
element.despues("h3" //selector o elemeto);
agregar
agrega el nuevo elemento al elemento dado como hijo
element.agregar("h3" //selector o elemeto);
prefijar
agrega el nuevo elemento como primer hijo del elemento dado
element.prefijar("h3" //selector o elemeto);

Contamos además con una función que nos permite remover elemetos y se llama remover 

S("mielemento").remover();

Tienes una para remplazar la cual remplaza un elemeto por otro 

element.remplazar("h3" //el elemento a remplazar);

Con clon podemos clonar elemetos y le pasamos un argumento boolean que por defecto es TRUE (verdadero) el cual establece si se debe clonar con sus hijo o false (falso) si solo se clona el elemento 

var mielemento = S("mielemento").clon();
/*
 * la variable mielemento ahora contine el clon del elemeto
 * seleccionado listo para ser modificado o introducido al HTML
 */

Para mover elementos ShellScript nos ofrece la función moverElem la cual recibe como único parametro el elemento de refencia 

S("mielemento").moverElem("h3" //selector o elemeto);
//el elemento sera movido despues del elemeto de refencia

Por último tenemos la función OuterElem la cual nos permite encerrar un elementos detro de otro. Los argumentos que recibe esta función son los mismo que dariamos si queremos crear un elemento nuevo con ShellScript 

S("h3").OuterElem("div",{
	parametros
});
//el h3 sera encerrado dentro de un nuevo div
