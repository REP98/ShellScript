Introducción

Bienvenidos al manual de ShellScript donde aprenderás cómo desarrollar con esta librería fácil y rápido.

ShellScript es un framework de JavaScript. Bien, y ¿qué es un framework?. Por su definición, es un macro de trabajo el cual establece una estructura útil y reusable la cual le permite desarrollar fácilmente sus aplicaciones. Entonces, ¿qué quiere decir esto?, que ShellScript es un macro de trabajo creado en lenguaje JavaScript el cual nos permite usar múltiples funciones de distintas maneras y que funcione incansablemente de tal forma que podamos reusar siempre.

ShellScript tiene múltiples ventajas a la hora de usarlo ya que es más rápido que otros framework y mas ligeros, posee muchas funciones en su código y permite al usuario modificarlo a sus necesidades. Para utilizarlo solo necesitas descargalo de esta página en la parte superior donde dice descarga o presiona segundo clic y en alert has clic donde dice aqui y listo. Una vez descargado creas tu página web.

Se te descargara un archivo comprimido .zip el cual debes descomprimilo y copiar la carpeta ShellScript completa al alojamiento de tu proyecto y ya.
Nota: el framework esta en trincado con la carpeta por tal motivo NO DEBEN sacar el js de hay deben llamarlo directamente dentro de la carpeta, ejemplo:<script src="tu_proyecto/ShellScript/ShellScript.js"></script>

ShellScript se usa de una manera similar a otros framework existentes en el mercado lo que hace este manual mas corto de explicar. además también posee un mini framework css3 con Responsive, animaciones y apariencia como la de esta página.
Si revisa el código de esta página verás que el css es mínino para la apariencia que tiene ya que todo es ShellScript.css
