Validadores

ShellScript posee múltiples funciones que validan el tipo de un objeto retornado verdadero o falso o si el existe, estos validadores son:

Classexiste
Valida si una clase pertenece a un elemento
enArray
Busca un valor en un array u objeto y retorna su posición en caso de encontrarlo en su defecto -1
esArray
Verifica si es un array
esBooleno
Valida si un valor es booleano (es decir TRUE o FALSE);
esElem
Verifica si es un elemento HTML
esFecha
Verifica si es una fecha o objeto DATE
esFuncion
Valida si es una función
esNumero
Verifica si es un número
esObj
Verifica si es un objeto
esRegExp
Verifica si es una expresion regular
esString
Valida si un valor es una cadena de texto;
esUndefined
Verifica si es indefinido
esVisible
Valida si un objeto dado es visible
esWindow
Verifica si es una ventana
existe
Verifica si un elemento existe
key_existe
Valida si existe un clave en un Array o en un objeto. si conocen PHP sabran que existe una función llamada array_key_exists, esta función es impirada el la de PHP
ObjLength
Obtiene el tamaño de un array u objeto

Veamos un ejemplos: 

S.existe("body"); //retorna TRUE

var a;
S.esUndefined(a); //retorna TRUE


S.esNumero(1);//retorna TRUE

S("h2").Classexiste("title");//retorna TRUE
S.Classexiste("div","subTitle");//retorna TRUE

var h = ["Shell","Script"];
var b = {Shell:"Script"};
S.enArray("Shell",h);//retorna 0
S.enArray("Script",b);//retorna Shell

var arrays = ["Shell","Script"];
var obj = {sript:"Shell",shell:"Script"};
S.key_existe(0,arrays);//retorna Shell
S.key_existe("sript",obj);//retorna Shell

var r= {a:0,b:1,c:2,d:3};
var t=["Shell","Script"];
S.ObjLength(t); //retorna 2 
S.ObjLength(r); //retorna 4
