ForEach

El ForEach es una de las funciones mas usadas en ShellScript y por ello es una de las más importantes a la hora de trabajar. te permite recorrer matrices, objetos y elementos; Ademas de que puedes asiganar parte o todo de una variable a otra. 

S.forEach(
	OBJ//Objeto, matriz o elemento,
	fn//una función para interactuar con el objeto,
	obj//Objeto que obtendra los valores del primer ojeto,
	rango//Array que contendra el inicio y fin que se va a recorre
);
//Si se va a recorrer un elemento se puede usar de esta forma
S("elemento").forEach(//Los mismo parametros de arriva omitiento el primero);
//Las funciones reciven 3 argumetos el this, el indice y el valor veamos un ejemplo
S("spna").forEach(function(i,v){
		console.log("Indice es: "+i,"El Valor es: "+v);
	}});
